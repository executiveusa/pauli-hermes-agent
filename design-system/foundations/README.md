# Design System Foundations

This folder contains the core design foundation files that define the visual language of the entire system.

## Files

- **colors.md** - Color palette, semantic roles, domain-specific guidelines
- **typography.md** - Typeface selection, type scale, font weights, line heights
- **spacing.md** - Spacing scale (8-point grid), usage principles, vertical/horizontal rhythm
- **shadows.md** - Shadow levels (soft, medium, strong, elevated) and usage rules
- **motion.md** - Animation principles, easing functions, interaction patterns
- **border-radius.md** - Radius scale and usage for different element types
