# Design System Layouts

This folder contains page layout templates and breakpoint strategies.

## Layout Types

- **single-column/** - Single column layouts (landing pages, blog)
- **two-column/** - Sidebar + main content layouts
- **three-column/** - Dashboard-style three-column layouts
- **grid-based/** - CSS Grid-based layouts
- **responsive/** - Mobile-first responsive examples

## Breakpoint Strategy

- **Mobile:** 0px - 640px
- **Tablet:** 640px - 1024px
- **Desktop:** 1024px - 1280px
- **Large Desktop:** 1280px+

## Mobile-First Principles

- Design for mobile first
- Progressive enhancement for larger screens
- No horizontal scroll on mobile
- Minimum 44px touch targets
