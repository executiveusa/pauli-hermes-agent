# Design System Patterns

This folder contains compound UI patterns and page-level patterns built from components.

## Pattern Categories

- **forms/** - Advanced form patterns (multi-step, conditional fields)
- **dashboards/** - Dashboard layouts and metric cards
- **sidebars/** - Sidebar navigation patterns
- **headers/** - Header variations (sticky, transparent, etc.)
- **footers/** - Footer patterns
- **modals/** - Modal composition patterns
- **notifications/** - Toast and notification patterns
- **loading-states/** - Page-level loading and skeleton patterns
