# Design Tokens

This folder contains exportable design tokens in multiple formats.

## Token Categories

- **colors.json** - Color palette and semantic roles
- **typography.json** - Typefaces, scale, weights, line heights
- **spacing.json** - Spacing scale values
- **tokens-index.md** - Token reference and usage guide

## Export Formats

- **css/** - CSS custom properties (--color-primary, --spacing-4, etc.)
- **json/** - JSON token structure for programmatic access
- **figma/** - Figma plugin-compatible token format

## Token Usage

All components must consume tokens via CSS variables or theme APIs. Do not hardcode values.
