# Design System Components

This folder contains all reusable UI components organized by category.

## Component Categories

- **buttons/** - Button variants (primary, secondary, tertiary, ghost)
- **inputs/** - Text inputs, selects, checkboxes, radio buttons
- **cards/** - Card container components
- **dialogs/** - Modal, alert dialog, sheet, drawer
- **navigation/** - Breadcrumbs, tabs, pagination, menu
- **forms/** - Form wrappers, field groups, validation
- **tables/** - Data table components
- **alerts/** - Alert, toast, notification components
- **badges/** - Badge and tag components
- **loading-states/** - Skeleton, spinner, progress components

## Component Structure

Each component folder contains:
- `component.tsx` - React component implementation
- `component.stories.tsx` - Storybook stories
- `README.md` - Documentation and usage guidelines
- `tests.tsx` - Component tests

## Library References

- **shadcn/ui** - Base primitives (buttons, inputs, dialogs, navigation)
- **cult-ui** - Elevated patterns and theming
- **motion-primitives** - Animation patterns
- **finance-bro** - Dashboard patterns (if applicable)
