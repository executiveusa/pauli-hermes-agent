# StoryToolkitAI - AI-Powered Video Production Toolkit

## Purpose
StoryToolkitAI is a comprehensive video indexing, transcription, and story development platform that enables AI agents to search through video footage, generate transcripts, translate content, and export timeline data for professional video editing workflows. It bridges the gap between raw video content and structured editing workflows using local AI models.

## Capabilities
- **Free Automatic Transcription**: Local Whisper integration for zero-cost transcription
- **AI-Powered Video Search**: Semantic search through indexed footage using embeddings
- **Multi-Language Translation**: Automatic translation to English from any language
- **Speaker Detection**: Automatic identification and labeling of speakers in footage
- **Story Editor**: Screenplay and script writing interface for narrative development
- **Timeline Export**: EDL, XML, and Fountain format exports for NLE integration
- **DaVinci Resolve Integration**: Direct project file compatibility with Resolve Studio
- **Transcript Management**: Group transcripts, detect questions, annotate content
- **Multi-Format Export**: SRT, TXT, AVID DS, Fusion Text+ output formats
- **Project Management**: Organize footage by projects, collections, and metadata
- **LLM Model Support**: Compatible with OpenAI, Ollama, vLLM, LM Studio

## Dependencies
- **Python 3.9+**: Core runtime environment
- **FFmpeg**: Video processing and frame extraction
- **Whisper**: Local speech-to-text transcription (faster-whisper recommended)
- **CUDA/GPU** (optional): Accelerated transcription and embedding generation
- **PostgreSQL or SQLite**: Metadata and index storage
- **Vector Store**: For semantic search capabilities (ChromaDB, Faiss, or pgvector)
- **LLM Backend**: OpenAI API, Ollama, vLLM, or LM Studio for AI features

## Usage
**For AI Agents:**
```python
# Initialize StoryToolkit
from storytoolkit import VideoIndexer, TranscriptGenerator

# Index video file
indexer = VideoIndexer(project="my_project")
video = indexer.add_video("path/to/video.mp4")

# Generate transcript
transcript = TranscriptGenerator.transcribe(video, model="whisper-large-v3")

# Semantic search
results = indexer.search("show me scenes with the red car", top_k=5)

# Export to editing software
indexer.export_timeline(results, format="fcpxml", output="timeline.xml")
```

**CLI Commands:**
```bash
# Index a video
storytoolkit index video.mp4 --project "Documentary"

# Transcribe with translation
storytoolkit transcribe video.mp4 --translate --language en

# Search footage
storytoolkit search "sunset beach scenes" --top 10

# Export EDL
storytoolkit export timeline.edl --format edl
```

## Integration
- **Video Editing**: Exports EDL/XML files compatible with DaVinci Resolve, Premiere Pro, Final Cut Pro, Avid Media Composer
- **Agent Zero**: Can be triggered by ClaudeCode or Switchblade for video analysis tasks
- **Byte Rover Memory**: Stores indexed metadata in `/memory/video-index/`
- **MCP Protocol**: Exposed as MCP tool for agent communication
- **Supermemory**: Can sync transcripts to memory system for long-term knowledge

## API/Interface
**REST API Endpoints:**
```
POST /api/videos/index          - Index new video file
POST /api/videos/transcribe     - Generate transcript
GET  /api/videos/search          - Semantic search
POST /api/timeline/export        - Export timeline data
GET  /api/projects               - List projects
GET  /api/transcripts/:id        - Get transcript
PUT  /api/transcripts/:id        - Update transcript annotations
```

**Python SDK:**
```python
from storytoolkit import StoryToolkit

# Initialize
toolkit = StoryToolkit(db="postgresql://localhost/storytoolkit")

# Main operations
toolkit.videos.index(path, metadata)
toolkit.transcripts.generate(video_id, model="whisper-large-v3")
toolkit.search.semantic(query, filters)
toolkit.timeline.export(clips, format, output)
toolkit.speakers.detect(video_id)
toolkit.projects.create(name, settings)
```

## Examples
**Example 1: Video Asset Search for Commercial Production**
```python
# Agent searches for specific B-roll footage
toolkit = StoryToolkit(project="Commercial_2026")
results = toolkit.search(
    query="product shots with dramatic lighting",
    filters={"duration": ">5s", "resolution": "4K"},
    top_k=20
)
# Export selected clips to editor
toolkit.export_timeline(results, format="fcpxml", output="broll_selects.xml")
```

**Example 2: Interview Transcription & Analysis**
```python
# Transcribe interview and detect speakers
video = toolkit.videos.add("interview.mp4")
transcript = toolkit.transcribe(video, detect_speakers=True)

# Find specific topics
qa_segments = toolkit.search_transcript(
    transcript_id=transcript.id,
    query="discussion about AI and ethics"
)

# Export segments with speaker labels
toolkit.export_transcript(qa_segments, format="srt", speakers=True)
```

**Example 3: Documentary Rough Cut Assembly**
```python
# Agent assembles rough cut based on narrative structure
story_beats = [
    "introduction to main character",
    "problem statement",
    "solution demonstration",
    "testimonials",
    "call to action"
]

timeline = []
for beat in story_beats:
    clips = toolkit.search(beat, top_k=3)
    timeline.extend(clips)

# Export to DaVinci Resolve
toolkit.export_timeline(timeline, format="resolve_xml", output="rough_cut.xml")
```

## Configuration
**Default Settings:**
```yaml
storage:
  database: "sqlite:///storytoolkit.db"  # or postgresql://
  video_cache: "./cache/videos"
  index_path: "./index"

transcription:
  model: "whisper-large-v3"
  language: "auto"
  translate: false
  device: "cuda"  # or cpu

search:
  embedding_model: "text-embedding-3-large"
  top_k: 10
  threshold: 0.7

export:
  default_format: "fcpxml"
  frame_rate: 24
  timecode_start: "00:00:00:00"
```

## Performance Notes
- Transcription: ~10-20x real-time on GPU, ~1-2x on CPU
- Indexing: 1-5 minutes per hour of footage (GPU-dependent)
- Search: Sub-second for datasets under 1000 videos
- Export: Near-instantaneous (metadata-only operations)

## Docker Integration
```dockerfile
FROM python:3.11-slim
RUN apt-get update && apt-get install -y ffmpeg
COPY requirements.txt .
RUN pip install storytoolkit whisper-faster
VOLUME ["/videos", "/index", "/output"]
CMD ["storytoolkit", "serve", "--host", "0.0.0.0"]
```

## File Locations
- **Project Root**: `StoryToolkitAI-main/`
- **Config**: `config/storytoolkit.yaml`
- **Cache**: `cache/videos/` (temporary frames and audio)
- **Index**: `index/` (vector embeddings and metadata)
- **Exports**: `exports/` (EDL, XML, SRT outputs)
- **Logs**: `logs/storytoolkit.log`

## Agent Interaction Patterns
```
User: "Find all scenes where we see the product being used"
↓
Agent Zero Routes to ClaudeCode
↓
ClaudeCode activates StoryToolkit Docker container
↓
StoryToolkit performs semantic search across indexed footage
↓
Returns timestamped clips with confidence scores
↓
ClaudeCode exports timeline to XML
↓
User imports XML into DaVinci Resolve
```
