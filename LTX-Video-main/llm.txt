# LTX-Video - DiT-Based Video Generation Model

## Purpose
LTX-Video is the first Diffusion Transformer (DiT) model for video generation that produces high-fidelity, synchronized audio-video content at up to 4K resolution and 50 FPS. It enables AI agents to generate professional-quality videos from text prompts, images, or other video inputs for creative production workflows.

## Capabilities
- **Text-to-Video Generation**: Create videos from natural language descriptions
- **Image-to-Video**: Animate static images with realistic motion
- **Video-to-Video**: Transform existing videos with new styles or attributes
- **Video Extension**: Extend videos forward or backward in time
- **Multi-Keyframe Conditioning**: Control animation with multiple reference frames
- **Synchronized Audio Generation**: Native audio-video synchronization
- **High Resolution Output**: Native 4K (3840x2160) video generation
- **High Frame Rate**: Up to 50 FPS for smooth motion
- **Control Models**: Depth, pose, and canny edge control for precise generation
- **LoRA Fine-Tuning**: Custom style and subject adaptation
- **Multiple Model Sizes**: 13B (quality) and 2B (speed) variants
- **Distilled Models**: Real-time generation with distilled checkpoints
- **ComfyUI Integration**: Node-based workflow for video generation
- **Diffusers Integration**: Python API for programmatic generation

## Dependencies
- **Python 3.10+**: Core runtime
- **PyTorch 2.0+**: Deep learning framework
- **CUDA 11.8+**: GPU acceleration (required)
- **GPU Memory**: 24GB+ VRAM for 13B model, 12GB+ for 2B model
- **Diffusers Library**: Hugging Face inference pipeline
- **Transformers**: Model loading and tokenization
- **ComfyUI** (optional): Visual workflow interface
- **FFmpeg**: Video encoding and compression
- **Torch Audio** (optional): For audio generation features

## Usage
**For AI Agents:**
```python
import torch
from diffusers import LTXVideoPipeline

# Initialize model
pipe = LTXVideoPipeline.from_pretrained(
    "Lightricks/LTX-Video",
    torch_dtype=torch.float16
).to("cuda")

# Text-to-video
prompt = "A drone shot flying over a misty forest at sunrise, cinematic, 4K"
video = pipe(
    prompt=prompt,
    num_frames=121,  # 5 seconds at 24fps
    height=768,
    width=1360,
    num_inference_steps=50
).frames

# Image-to-video
from PIL import Image
init_image = Image.open("landscape.jpg")
video = pipe(
    prompt="Camera slowly pans left",
    image=init_image,
    num_frames=121,
    strength=0.8
).frames

# Save output
from diffusers.utils import export_to_video
export_to_video(video, "output.mp4", fps=24)
```

**CLI Commands:**
```bash
# Generate video from prompt
python generate.py --prompt "A cat playing piano" --output cat_piano.mp4

# Image-to-video with control
python i2v.py --image input.jpg --prompt "Zoom in slowly" --control depth

# Video extension
python extend.py --input video.mp4 --extend forward --frames 60

```

## API/Interface
**Python SDK:**
```python
from ltxvideo import LTXVideo

# Initialize generator
generator = LTXVideo(model="ltxv-13b", device="cuda")

# Generate with advanced options
video = generator.generate(
    prompt="Cinematic shot of New York City at night",
    negative_prompt="blurry, low quality, distorted",
    num_frames=121,
    resolution=(768, 1360),
    fps=24,
    guidance_scale=7.5,
    num_steps=50,
    seed=42
)

# Keyframe animation
keyframes = [
    {"frame": 0, "image": "start.jpg"},
    {"frame": 60, "image": "mid.jpg"},
    {"frame": 120, "image": "end.jpg"}
]
video = generator.keyframe_interpolate(keyframes, prompt="Smooth transition")

# Video-to-video with control
video = generator.v2v(
    input_video="source.mp4",
    prompt="Convert to anime style",
    control_type="canny",
    control_strength=0.8
)
```

## Examples

```python
# Cynthia (Designer) generates hero video for landing page
generator = LTXVideo(model="ltxv-13b")

Professional product photography style, 
luxury watch on marble surface, 
dramatic side lighting,
```

**Example 2: B-Roll Generation for Documentary**
```python
# ClaudeCode generates supplementary footage
scenes = [
    "Time-lapse of clouds over mountains, golden hour",
    "Close-up of rain drops on window, shallow depth of field",
    "Aerial view of winding highway through desert landscape",
    "Macro shot of circuit board with pulsing lights"
]

broll_clips = []
for i, scene in enumerate(scenes):
    video = generator.generate(
        prompt=f"{scene}, cinematic, professional grade",
        num_frames=73,  # 3 seconds each
        resolution=(768, 1360)
    )
    filename = f"broll_{i:03d}.mp4"
    generator.save(video, filename)
    broll_clips.append(filename)
```

## Configuration
**Model Selection:**
```yaml
models:
  ltxv-13b:
    quality: highest
    vram: 24GB
    speed: ~2 min/video
    
  ltxv-2b:
    quality: good
    vram: 12GB
    speed: ~30 sec/video
    
  ltxv-13b-distilled:
    quality: high
    vram: 16GB
    speed: ~45 sec/video
```

**Generation Settings:**
```yaml
generation:
  resolution:
    landscape: [768, 1360]    # 16:9
    vertical: [1080, 1920]    # 9:16
    square: [1080, 1080]      # 1:1
  
  fps: [24, 30, 50]
  
  quality:
    draft: {steps: 20, guidance: 6.0}
    standard: {steps: 50, guidance: 7.5}
    high: {steps: 75, guidance: 8.5}
    
  defaults:
    num_frames: 121  # 5 seconds at 24fps
    guidance_scale: 7.5
    num_inference_steps: 50
```

## Performance Notes
- **13B Model**: ~2 minutes per 5-second video on A100 GPU
- **2B Model**: ~30 seconds per 5-second video on A100 GPU
- **Distilled Model**: ~45 seconds per 5-second video on A100 GPU
- **VRAM Usage**: 13B requires 24GB, 2B requires 12GB
- **Batch Generation**: Supported with minimal overhead (linear scaling)
- **CPU Generation**: Not recommended (hours per video)

## LoRA Fine-Tuning
```python
# Train custom LoRA for brand-specific style
from ltxvideo.training import LoRATrainer

trainer = LoRATrainer(base_model="ltxv-13b")
trainer.load_dataset("brand_videos/", captions="captions.txt")
trainer.train(
    output_dir="lora_brand_style",
    learning_rate=1e-4,
    num_epochs=100,
    rank=64
)

# Use fine-tuned LoRA
generator = LTXVideo(model="ltxv-13b", lora="lora_brand_style")
video = generator.generate("Product showcase in our brand style")
```
