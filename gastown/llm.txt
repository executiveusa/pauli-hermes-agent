## Purpose
Gastown is the primary orchestration framework for coordinating multiple AI agents in complex, multi-step workflows with state management and failure recovery.

## Core Capabilities
- **Task Routing:** Intelligent distribution based on agent capabilities
- **Agent Coordination:** Synchronized execution across multiple agents
- **Workflow Management:** DAG-based workflow definitions
- **State Management:** Persistent workflow state with rollback
- **Failure Recovery:** Automatic retry with exponential backoff
- **Load Balancing:** Distribute work across agent instances

## Workflow Patterns

### Sequential
```yaml
workflow:
  name: "Full-Stack Feature"
  pattern: sequential
  steps:
    - agent: claudecode
      task: "Design API endpoints"
    - agent: cynthia  
      task: "Design UI components"
    - agent: switchblade
      task: "Deploy to staging"
```

### Parallel
```yaml
workflow:
  pattern: parallel
  branches:
    - agent: storykit
      task: "Transcribe video"
    - agent: memory
      task: "Index content"
  merge_strategy: wait_all
```

### Conditional
```yaml
workflow:
  pattern: conditional
  condition: "complexity > 8"
  then:
    agent: claudecode
    model: "claude-opus"  # Use powerful model
  else:
    agent: claudecode
    model: "glm-4"  # Use cost-efficient model
```

### Loop
```yaml
workflow:
  pattern: loop
  condition: "tests_passing == false"
  max_iterations: 5
  steps:
    - agent: claudecode
      task: "Fix failing tests"
    - agent: switchblade  
      task: "Run test suite"
```

## Agent Communication
Gastown uses the Agent Mail protocol for inter-agent communication:

```python
from gastown import Workflow, AgentTask

workflow = Workflow(name="feature_development")

# Define tasks
task1 = AgentTask(
    agent="claudecode",
    action="generate_code",
    input={"feature": "user authentication"}
)

task2 = AgentTask(
    agent="cynthia",
    action="design_ui",
    input={"feature": "login form"},
    depends_on=[task1]  # Wait for task1
)

# Execute workflow
result = workflow.execute([task1, task2])
```

## State Management
```python
# Save workflow state
workflow.save_checkpoint("checkpoint_1")

# Resume from checkpoint
workflow = Workflow.load("workflow_id", "checkpoint_1")
workflow.resume()

# Rollback on failure
workflow.rollback_to("checkpoint_1")
```
