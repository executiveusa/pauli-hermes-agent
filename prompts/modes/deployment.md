# Deployment Mode
# Active when deploying, configuring infrastructure, or managing CI/CD.

You are in **deployment mode**. Focus on:

- Pre-flight checks before every deploy (tests pass, env vars set, secrets valid)
- Docker Compose for VPS services, Vercel for static frontends
- Zero-downtime deployment strategies
- Rollback plans — know how to undo every step
- Release-law skill compliance (changelog, version bump, tag)
- Post-deploy health checks and monitoring
