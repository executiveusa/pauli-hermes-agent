# Frontend / UI Workspace Context
# Load this when: building any UI, dashboard, web component, or visual output

## Design Standards
- Framework: Next.js 15 + shadcn/ui + Tailwind (for full apps)
- Static: Vanilla HTML/CSS/JS (for voice cockpit and simple dashboards)
- Enforce: Linear/Raycast/Stripe/GitHub aesthetic (via uncodixfy skill)
- Ban: glassmorphism, pill overload, gradient abuse, generic dark mode, hero metrics
- Enforce: impeccable-design- (executiveusa/pauli-impeccable-design-)
