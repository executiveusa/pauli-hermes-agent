# Brand Law

## Purpose
Ensures every client-facing output adheres to brand identity standards.

## Rules

1. **Logo usage**: Never modify, distort, or place logos on busy backgrounds. Minimum clear space = logo height.
2. **Color palette**: Use only approved brand colors. Check the client's brand guide or Open Brain (collection: "brands") for palettes.
3. **Typography**: Use specified typefaces. If unspecified, default to Inter (body) + Space Grotesk (headings).
4. **Voice**: Match the client's documented tone. If unknown, default to "professional yet approachable."
5. **Positioning**: All messaging must align with the brand's stated positioning and values.
6. **Consistency**: Cross-platform outputs must be visually and tonally consistent.

## Checklist
Before delivering any branded output:
- [ ] Colors match brand palette
- [ ] Typography follows brand guide
- [ ] Tone matches brand voice
- [ ] Logo usage follows guidelines
- [ ] No competing brand elements present
