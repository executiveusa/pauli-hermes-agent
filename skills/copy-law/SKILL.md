# Copy Law

## Purpose
Ensures all written content meets quality, clarity, and brand voice standards.

## Rules

1. **Clarity**: One idea per sentence. No jargon unless the audience expects it.
2. **Concision**: Cut every word that doesn't earn its place. Target 50% fewer words than first draft.
3. **Active voice**: Default to active voice. Passive only when the actor is irrelevant or unknown.
4. **Structure**: Lead with the most important information. Use inverted pyramid for all non-narrative content.
5. **Headings**: Headings are promises — the content below must deliver what the heading claims.
6. **CTAs**: Every action-oriented piece needs one clear call to action. Never more than two.
7. **Tone matching**: Match the client's documented voice. When unspecified, use "confident, clear, human."
8. **No filler**: Ban "leverage", "utilize", "in order to", "it should be noted that", "please note".

## Checklist
- [ ] Active voice throughout
- [ ] No filler words/phrases
- [ ] One clear CTA per action piece
- [ ] Headings match content below them
- [ ] Tone matches brand voice guide
