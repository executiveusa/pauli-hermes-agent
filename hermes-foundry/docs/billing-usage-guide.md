# Billing & Usage Guide

Usage endpoint: `GET /v1/usage/{tenant_id}`.

Tracked dimensions:
- tokens_in / tokens_out
- provider cost estimate
- browser/coding minutes
- storage usage
- workflow count
- sub-agent count
- API/MCP call counts

Billing plans:
- free
- pay_as_you_go
- sprint
- managed_retainer
- internal_nonprofit_discount
- manual_invoice_only
